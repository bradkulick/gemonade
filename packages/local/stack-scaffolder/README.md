# Stack Scaffolder

A Gemonade Gem for rapidly bootstrapping new projects.

## Usage

1. Load the Gem:
   ```bash
   gemonade stack-scaffolder
   ```

2. State your requirements:
   > "Create a Python FastAPI project with Docker support."
   > "Scaffold a React app with Vite and Tailwind."
   > "I need a basic Go CLI tool structure."

3. Review the proposed tree.
4. Confirm to generate files.
